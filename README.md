# Medical Treatment Planner AI

## Project Description
Medical Treatment Planner is an AI-based project developed using Python and Machine Learning for treatment planning and healthcare management.

## Features
- Treatment planning
- Medical record management
- AI-based suggestions
- Schedule management

## Technologies Used
- Python
- Machine Learning
- Flask
- HTML
- CSS

## How to Run
1. Install required libraries
2. Run app.py
3. Open in browser
